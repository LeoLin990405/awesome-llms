# Contributing to Awesome LLMs / 贡献指南

Thank you for your interest in contributing! This guide explains how to suggest new entries or improvements.

感谢您有兴趣为本项目做出贡献！本指南将说明如何建议新条目或改进。

---

## How to Add a New Entry / 如何添加新条目

### 1. Check for Duplicates / 检查是否重复

Search the existing README to make sure the model, tool, or resource is not already listed.

在提交之前，请先搜索 README 以确保该模型、工具或资源尚未收录。

### 2. Fork and Edit / Fork 并编辑

1. Fork this repository / Fork 本仓库
2. Create a new branch: `git checkout -b add-<entry-name>`
3. Edit `README.md` and add your entry in the **correct category**
4. Maintain **alphabetical order** within each table
5. Commit and push your changes

### 3. Follow the Bilingual Format / 遵循双语格式

All entries must include both English and Chinese descriptions:

所有条目必须包含中英文描述：

**For models / 模型格式:**
```markdown
| [Model Name](https://link) | Organization | Parameters | Description in English / 中文描述 | ![Stars](https://img.shields.io/github/stars/owner/repo) |
```

**For tools / 工具格式:**
```markdown
| [Tool Name](https://link) | Description in English / 中文描述 | ![Stars](https://img.shields.io/github/stars/owner/repo) |
```

### 4. Submit a Pull Request / 提交 PR

- Use a clear, descriptive title (e.g., "Add DeepSeek-V4 to Open-Source Models")
- Explain why this entry belongs in the list
- One entry per PR is preferred, but bulk additions in the same category are fine

---

## Quality Standards / 质量标准

Your suggested entry must meet these criteria:

建议的条目必须满足以下标准：

| Criterion / 标准 | Requirement / 要求 |
|---|---|
| **Relevance / 相关性** | Must be related to LLMs (models, tools, benchmarks, resources) |
| **No duplicates / 不重复** | Not already in the list |
| **Working links / 有效链接** | All URLs must be accessible |
| **Active maintenance / 积极维护** | For open-source: commit within the last 6 months |
| **GitHub stars badge / 星标徽章** | Required for open-source entries with a GitHub repo |
| **Bilingual / 双语** | Both English and Chinese descriptions |

---

## Reporting Issues / 报告问题

Found a broken link, outdated info, or miscategorized entry? Please:

发现失效链接、过时信息或分类错误？请：

1. [Open an issue](https://github.com/LeoLin990405/awesome-llms/issues/new) describing the problem
2. Or submit a PR with the fix directly

---

## Code of Conduct / 行为准则

- Be respectful and constructive / 尊重他人，建设性交流
- No promotional spam / 禁止推广性垃圾信息
- Focus on quality over quantity / 重质量而非数量

---

Thank you for helping make this list better!

感谢您帮助完善这份列表！
